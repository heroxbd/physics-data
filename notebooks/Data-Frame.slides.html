GNU R¶

  • R 是著名的统计学语言，应用广泛
  • 起源于1970年代的 S 语言
    • 在 fortran 库的基础上交互地处理数据进行统计分析
      • 调用 fortran 库提供交互环境是 S, Matlab 和 Numpy 的共同起源
    • 提供交互式的绘图工具，以便于快速迭代
    • 受 LISP 语言的影响
    • Data Frame 概念影响了几乎所有统计工具
    • S-PLUS, SAS, SPSS 应用广泛
  • GNU R 重新使用 Scheme 实现了 S 语言，并以自由软件形式发布
    • 自由软件和开源运动是为科学研究提供了可 复现 的工具

为什么使用 R¶

  • 大部分统计学算法都有 R 语言的实现

    • 非常适合快速试验，找到统计方法上的 最佳工具

  • dplyr：数据整理的利器

    • DataFrame 上的高级关系代数运算
    • https://dplyr.tidyverse.org/

  • data.table

    • 高性能的 DataFrame 实现

  • 在 Jupyter 上运行 R，IRKernel

    • 与 Jupyter Python 有同样的功能，Jupyter 可以驱动各类语言
    • https://github.com/IRkernel/IRkernel

  • 安装：

    • apt install r-cran-dplyr r-cran-data.table r-cran-irkernel
