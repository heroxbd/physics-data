数据的可视化¶

  • Matplotlib

    • 数据可视化、画图工具，风格受 Matlab 影响
    • 参考资料：Scipy Lecture Notes

  • X Window System

    • 也被称为 X11
    • 图形显示的国际标准

X11¶

  • Server-client 架构
  • Server 为显示图形提供服务
  • Client 连接 X11 server，请求显示图形

配置环境¶

  • Windows

    • 下载并安装 VcXsrv
    • VcXsrv 是使用 Visual Studio 编译的 X11
    • X11 server 运行在 Windows
      • X11 client 在 WSL 中
      • 两者通过本地网络接口通信

  • MacOS

    • 默认无 X11 环境
    • 但是众多程序可以直接在 MacOS 的图形界面输出

  • GNU/Linux

    • 默认有 X11 环境

画图原理¶

  • 画图容易，但是画出有信息量，突出本质，令人赏心悦目的好图不易。
  • 画图有艺术成分，没有一定之规，只能积累经验逐步提升
  • 但有一些基本原理需要注意，掌握之后可以满足基本要求

参考：A layered grammar of graphics by Hadley Wickham

图是给人类看的¶

  • 字要让人类看清
  • 颜色对比在鲜明，易于人类辩认
  • 要有可以让人类读懂的标识
